# Chain Rule

# Core Formula

dxd​[f(g(x))]=f′(g(x))⋅g′(x)

Or in Leibniz notation:

dxdz​=dydz​⋅dxdy​

# Why It Matters for ML

  • Foundation of backpropagation — gradients flow backward through composed functions
  • Each layer in a neural net is a function composition → chain rule applied repeatedly

# ⚠️ Tricky Interview Facts

  • Vanishing gradients are a direct consequence of chaining many small derivatives (e.g., sigmoid saturates → σ'(x) ≈ 0 → gradient collapses)
  • In vector form, the scalar chain rule becomes a product of Jacobian matrices — order matters (matrix multiplication is not commutative)
  • During backprop, intermediate activations must be cached from the forward pass to compute the chain rule — this is why memory scales with depth
