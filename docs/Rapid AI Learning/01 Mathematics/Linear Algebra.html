Linear Algebra

# Basis Vectors

Definition: A set of vectors that is linearly independent and spans the vector space.

  • Minimum set needed to represent every vector in the space
  • Basis is not unique; standard basis is just the most common choice
  • Number of basis vectors = dimension of the space

Standard basis in ℝⁿ: e₁=(1,0,…,0), e₂=(0,1,…,0), …, eₙ=(0,0,…,1)

Any vector x ∈ ℝⁿ can be written uniquely as:

x=i=1∑n​ci​ei​

# Span of Vectors

Definition: The span of a set of vectors {v₁, v₂, …, vₙ} is the set of all possible linear combinations:

span{v1​,...,vn​}={i=1∑n​ci​vi​∣ci​∈R}
  • Geometrically: the subspace “reachable” by scaling and adding those vectors
  • span of 1 vector = line; 2 non-collinear vectors = plane; etc.

# Linear Transformations and Matrices

  • A linear transformation maps vectors to vectors, preserving addition and scalar multiplication
  • Every linear transformation can be represented by a matrix
  • Columns of the matrix = where basis vectors î, ĵ land after transformation
  • Types: rotation, shear, scaling, projection, reflection

# Matrix Multiplication as Composition

  • AB = apply B first, then A (right-to-left order)
  • Not commutative: AB ≠ BA in general
  • Associative: (AB)C = A(BC)
  • Each output column = matrix A applied to corresponding column of B

# Formula — general element (core rule)

(AB)ᵢⱼ = Σₖ Aᵢₖ · Bₖⱼ  =  row i of A  ·  col j of B
  • A is (m×p), B is (p×n) → AB is (m×n). Inner dims must match.

# 2×2 explicit expansion

| a  b | | e  f |   | ae+bg  af+bh |
| c  d | | g  h | = | ce+dg  cf+dh |

# Complexity

O(m · p · n) multiplications for (m×p)(p×n)
  • Naive. Strassen: O(n^2.807). Practical: use BLAS/cuBLAS.

# Dot Product

# Algebraic (component-wise)

a · b = Σᵢ aᵢbᵢ = a₁b₁ + a₂b₂ + ··· + aₙbₙ
  • Result is a scalar. Order doesn’t matter: a·b = b·a

# Geometric (angle-based)

a · b = |a| |b| cos θ
  • θ = angle between vectors. Zero when perpendicular (orthogonal).

# Matrix form

a · b = aᵀb
  • Row vector × column vector → (1×n)(n×1) = scalar
  • Relevance: attention scores = QKᵀ is a batch of dot products → similarity measurement

# Inverse Matrices, Column Space and Null Space

Concept Definition
Inverse A⁻¹ AA⁻¹ = I; exists only if det(A) ≠ 0
Column space Span of all possible outputs Ax (range)
Null space / Kernel All x where Ax = 0
Rank Dimension of column space
  • Rank-nullity theorem: rank(A) + nullity(A) = n (number of columns)

# Inverse — definition

A A⁻¹ = A⁻¹ A = I

Exists iff det(A) ≠ 0 (A is square and full rank).

# 2×2 closed form

| a  b |⁻¹       1      |  d  -b |
| c  d |   =  ———————— · | -c   a |
               ad - bc

# Product rule

(AB)⁻¹ = B⁻¹A⁻¹

Order reverses — same rule as transpose: (AB)ᵀ = BᵀAᵀ

# Column space / range

Col(A) = { Ax | x ∈ ℝⁿ }

Span of all outputs. dim = rank(A).

# Null space / kernel

Null(A) = { x | Ax = 0 }

dim = nullity(A). Non-trivial when rank < n.

# Nonsquare Matrices as Transformations Between Dimensions

  • m×n matrix: transforms from ℝⁿ → ℝᵐ
  • n > m → dimension reduction (e.g., embeddings)
  • n < m → maps into higher-dimensional space
  • Can never be invertible (no square inverse exists)

# Determinant

# 2×2

det | a  b | = ad - bc
    | c  d |

# 3×3 (cofactor expansion)

det(A) = a₁₁(a₂₂a₃₃ - a₂₃a₃₂)
       - a₁₂(a₂₁a₃₃ - a₂₃a₃₁)
       + a₁₃(a₂₁a₃₂ - a₂₂a₃₁)

# Key identities

det(AB) = det(A) · det(B)
det(Aᵀ) = det(A)
  • det = signed volume scaling factor of the transformation

# Dot Products and Duality

  • a · b = |a||b|cos θ = Σaᵢbᵢ
  • Geometric meaning: projection of b onto a, scaled by |a|
  • Duality: a 1×n matrix (row vector) is the “dual” of a column vector — dot product = matrix-vector multiplication
  • Relevance: attention scores = QKᵀ is a batch of dot products → similarity measurement

# Eigenvectors and Eigenvalues

Definition: Av = λv — vector v only scales (doesn’t rotate) under A

# Key formulas

det(A − λI) = 0          ← solve for eigenvalues λ
(A − λI)v = 0            ← plug λ back in to solve for eigenvectors v

# Trace & determinant identities

tr(A)   = Σᵢ λᵢ          ← sum of eigenvalues
det(A)  = Πᵢ λᵢ          ← product of eigenvalues

# Diagonalization

A = P Λ P⁻¹
  • P = eigenvector columns, Λ = diagonal eigenvalue matrix
  • Requires n independent eigenvectors

# Properties

  • Symmetric matrices → always real eigenvalues, orthogonal eigenvectors
  • Diagonal matrix → eigenvalues are the diagonal entries
  • det(A) = product of eigenvalues; tr(A) = sum of eigenvalues

# ML relevance

Application Role of Eigenvectors/values
PCA Eigenvectors of covariance matrix = principal components
SVD U, Vᵀ columns are eigenvectors of AAᵀ, AᵀA
Graph NNs Laplacian eigenvectors define graph structure
PageRank Dominant eigenvector of adjacency matrix

# Rank–Nullity & Subspaces

# Rank–nullity theorem

rank(A) + nullity(A) = n     (n = number of columns of A)

# Norms

# Vector norms

‖x‖₂ = √(Σ xᵢ²)           ← L2 / Euclidean
‖x‖₁ = Σ |xᵢ|              ← L1 / Manhattan
‖x‖∞ = max |xᵢ|            ← L∞ / Chebyshev

# Frobenius norm (matrix)

‖A‖_F = √(Σᵢⱼ aᵢⱼ²) = √(tr(AᵀA))

Most common matrix norm in ML (e.g. weight regularization).

# Tricky Interview Facts

  1. A matrix can have complex eigenvalues even with all real entries (e.g., rotation matrix)
  2. Eigenvectors are not unique — any scalar multiple is also an eigenvector; only the direction matters
  3. In PCA, you sort by largest eigenvalue, not smallest — variance explained is proportional to λ, not 1/λ
  4. Matrix multiplication is not commutative: AB ≠ BA in general, but it is associative
  5. A non-square matrix can never be invertible — no square inverse exists
  6. det = 0 means the transformation collapses space into a lower dimension (rank-deficient)
  7. Dot product = 0 means orthogonality, not that either vector is zero
